ⓘ This interactive app demonstrates a function that finds a circle that is tangent to, in between of, three other circles. The function will be a part of a circle packing algorithm. The resulting circle packing can then be applied to data visualization. The function finds a position and radius for a circle (gray), so that it becomes tangent to three known circles (red, green, blue). The known circles can be dragged around by mouse. Some ChatGPT was used to generate an empty canvas app. Akseli Palén, 2024.
